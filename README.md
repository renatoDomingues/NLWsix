
## NWL 'Next Level Week' da RocketSeat
````
Projeto desenvolvido para gerenciar perguntas anonimas de usuários direcionadas ao dono da página. Projeto desenvolvido na semana NLW 'Next Level Week' da RocketSeat na missão discover, ao sobre comando da Jakeliny!

Project developed to manage anonymous questions from users directed to the owner of the page. Project developed during the NLW 'Next Level Week' by RocketSeat on the discover mission, under command of Jakeliny!
````

### INSTALL

````
- npm init -y
- npm install ejs
- npm install express
- npm install nodemon -D
````

````
"start": "node src/server.js"
"dominguesDEV": "node src/server.js"

````
